# 02_applications — The Products

**Layer 4 (working artifacts).** One subfolder per company, each a complete tailored application.

## Naming
`YYYY-MM-Company-RoleTitle/`

## Each folder contains
- `brief.md` — score + fit map + submission rules
- `cv.md` — ATS-tailored CV (reads facts from `00_master/`)
- `letter.md` — tone-matched cover letter
- `form-answers.md` — salary / notice / authorisation, with [FILL IN]s
- `final/` — exported PDFs

## Rules
- New applications ONLY here, one subfolder per company.
- The pipeline creates these; you review and fill the flagged [FILL IN]s before submitting.
- Facts come from `00_master/` only. Nothing invented.
